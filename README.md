## Purpose

This repository contains the public-facing organization profile and default community health files for the Shine Harvest GitHub organization. It is the only public repository in the organization — all 10 program repositories are private and proprietary.

## Contents

### Organization Profile

| File | Purpose |
|:-----|:--------|
| `profile/README.md` | Organization overview displayed at [github.com/shineharvest](https://github.com/shineharvest) |
| `profile/logo-light.png` | Logo variant for light mode (transparent background, dark gold) |
| `profile/logo-dark.png` | Logo variant for dark mode (transparent background, bright gold) |

### Community Health Files

These files apply as defaults to all repositories in the organization that do not have their own versions.

| File | Purpose |
|:-----|:--------|
| `CONTRIBUTING.md` | Org-level contribution guidelines — team membership, branch model, squash merge, state dictionary compliance |
| `SECURITY.md` | Vulnerability reporting process, severity classification, automatic Critical classification for beam-safety items |
| `CODE_OF_CONDUCT.md` | Professional conduct standards, technical integrity, confidentiality, safety culture |

## Program Repositories

All program repositories are private. For the full repository map, architecture diagram, and milestone status, see the [organization profile](https://github.com/shineharvest).

| Phase | Repositories |
|:------|:-------------|
| A — Governance | sh-architecture, sh-simulations, sh-verification, sh-security |
| B — Physical Truth | sh-solar-generation, sh-orbital-platform |
| C — Operations | sh-beam-control, sh-ground-segment, sh-autonomy |
| D — Value | sh-market-layer |

## Important

This is a controlled engineering program. This public repository contains no proprietary technical content — only organizational governance defaults and the public profile page. All engineering content resides in the private program repositories.
